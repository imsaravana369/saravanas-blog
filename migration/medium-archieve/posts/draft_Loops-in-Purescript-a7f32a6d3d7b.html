Loops in Purescript

There were times I used loops for almost everything, <give-some-example>. But after starting to code in functional programming, they are mostly replaced with recursive function call. I’m not telling recursion is bad, but where are the cute loops is it even possible in functional programming? 

Yes my dear so friends its possible and there are different types of loops allowing different level of context.

Let’s start with loops that are pure,

  1. Pure While Loop

Sometimes we just want to apply a certain function to a variable until a predicate fails to hold.

E.g Doubling a number (starting from K) unless its less than N

while :: forall a. (a -> Boolean) -> (a -> a) -> a -> a
  • The first argument is the predicate function which decides whether the loop will run or not.
  • The second argument is the function that generates the next value from the previous value
  • The third argument is the initial value.

Example: 

Doubling a number until its less than N

>>> while (\curr -> curr < 100) (\prev -> 2 * prev) 1 
>>> 128
Note: While is not defined in standard prelude modle, we have to import it from third party package called purescript-Neon .
