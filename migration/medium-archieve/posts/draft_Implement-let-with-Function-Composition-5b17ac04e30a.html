Implement let with Function Composition

So, lets take the above example

myComposedFunction initialValue = 
     let firstResult = firstFunction initialValue
         secondResult = secondFunction firstResult
         thirdResult = thirdFunction firstResult secondResult
      in 
         thirdResult

this can be written into

composedFunction initialValue =   
                   (\firstResult -> 
                    (\secondResult -> 
                          thirdFunction secondResult result2) 
                         $ secondFunction firstResult) 
                         $ firstFunction initialValue

If you are not from functional programming side, then the below JS code might help,

var composedFunction =  (initialValue) => {
      return ((firstResult) => {
          return ((secondResult) => {
              return thirdFunction(firstResult,secondResult);
            }) (secondFunction(firstResult));
      }) (firstFunction(initialValue));
}
